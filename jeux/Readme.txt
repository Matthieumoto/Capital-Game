Comment utiliser notre jeu, c'est très simple :

  - Premièrement, vous devez avoir python d'installé sur votre ordinateur (python 3.10 de préference)
  - Ensuite, exécutez le modules.bat pour avoir les modules nécessaires à l'exécution de notre jeu (pygame, flask)
  - Enfin, Exécutez le main.py et AMUSEZ VOUS :) 

  Une fois dans le jeu vous avez le choix entre 4 options :
    - Jouer --> Jouer au jeu.
    - Info --> Un menu qui vous explique les règles du jeu.
    - Quitter --> Quitter le jeu.
    - Score --> Vous redirige ver un site de qualité pour afficher les scores entre les joueurs

  Pour jouer vous devez utiliser les touches directionel du clavier (Haut, Bas, Droite, Gauche).
  L'objectif est de récupérer 100 000 $ le plus vitre pour gagner le respect de vos amis.



/!\ Ne pas lire : 

Code de triche -->  "c, haut, haut, bas, bas, gauche, droite, gauche, droite, b, a"

/!\ Attention aux oreilles, une musique de qualite

Code --> "c, m, o, r, t"
